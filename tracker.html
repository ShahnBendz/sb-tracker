☁️ Google Sheets Setup

Connect to Google Sheets so your data syncs across every device. Takes about 5 minutes to set up once.

Setup Steps
1. Open Google Sheets → create a new blank sheet
2. Click Extensions → Apps Script
3. Delete all existing code, paste the script from google-apps-script.js (on your Desktop)
4. Click Deploy → New Deployment → Web App
5. Set Execute as: Me · Who has access: Anyone
6. Click Deploy → copy the Web App URL
7. Paste it below and click Save
